Frontend

pnpm이란 무엇인가?

공유 저장소와 심볼릭 링크 구조를 사용해 빠르고 디스크 효율적인 패키지 설치를 제공하는 패키지 매니저입니다.

패키지 매니저

pnpm이란 무엇인가?

pnpm은 npm과 같은 Node.js 패키지 매니저이지만, 패키지를 전역 content-addressable store에 한 번 저장하고 프로젝트에는 링크로 연결하는 방식이 특징입니다. 같은 패키지를 여러 프로젝트에서 중복 저장하지 않아 디스크를 아낄 수 있습니다.

간단한 예시

pnpm install
pnpm add axios
pnpm dev
pnpm build

특징

특징설명
빠른 설치이미 저장된 패키지를 재사용해 설치가 빠릅니다.
디스크 절약공유 저장소를 사용해 중복 저장을 줄입니다.
엄격한 의존성선언하지 않은 패키지를 우연히 import하는 문제를 줄입니다.
lockfilepnpm-lock.yaml로 설치 결과를 고정합니다.

기술 포인트

  • pnpm이 빠른 이유는?패키지를 공유 저장소에 저장하고 프로젝트에는 링크를 구성해 같은 파일 다운로드와 복사를 줄이기 때문입니다.
  • npm과 호환되나?대부분의 npm 패키지를 사용할 수 있지만, 엄격한 의존성 구조 때문에 숨은 의존성 문제가 드러날 수 있습니다.
