Vue3

watch와 watchEffect 차이

watch는 어떤 값을 볼지 명시하고 이전 값과 새 값을 받을 수 있습니다. watchEffect는 콜백 안에서 사용한 반응형 값을 자동으로 추적해 즉시 실행됩니다.

예시

watch(userId, (id, oldId) => fetchUser(id))

watchEffect(() => {
  console.log(userId.value, filter.value)
})

기술 포인트

  • 정확한 감시가 필요하면?watch가 적합합니다. 대상이 명확하고 이전 값 비교가 가능합니다.
  • 자동 추적이 편한 경우는?여러 반응형 값을 함께 읽는 간단한 effect에는 watchEffect가 편합니다.
