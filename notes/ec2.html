EC2란 무엇인가?

AWS에서 필요한 사양의 가상 서버를 생성하고 운영할 수 있게 해주는 컴퓨팅 서비스입니다.

AWS

EC2란 무엇인가?

EC2(Elastic Compute Cloud)는 AWS에서 제공하는 가상 서버 컴퓨팅 서비스입니다. 애플리케이션 서버, 배치 서버, 테스트 서버 등으로 사용할 수 있습니다.

핵심 개념

항목설명
정식 명칭Amazon Elastic Compute Cloud
서비스 유형IaaS, 가상 서버
기본 단위EC2 Instance
주요 선택 요소AMI, 인스턴스 타입, 스토리지, 보안 그룹, 키 페어
대표 사용처웹 서버, WAS, 배치 서버, 테스트 서버

EC2 구성 요소

구성 요소설명
AMI인스턴스 생성에 사용하는 OS/소프트웨어 이미지입니다.
Instance TypeCPU, 메모리, 네트워크 성능 등 서버 사양을 결정합니다.
EBSEC2에 연결하는 블록 스토리지입니다.
Security Group인바운드/아웃바운드 트래픽을 제어하는 가상 방화벽입니다.
Key PairSSH 접속 인증에 사용하는 공개키/개인키 쌍입니다.
Elastic IP고정 공인 IP 주소입니다.
