Pinia란 무엇인가?

Vue 애플리케이션에서 여러 컴포넌트가 공유하는 상태를 관리하는 공식 상태 관리 라이브러리입니다.

Vue3

Pinia란 무엇인가?

Pinia는 Vue의 상태 관리 라이브러리입니다. 로그인 사용자, 장바구니, 전역 설정처럼 여러 화면에서 공유해야 하는 상태를 store로 관리합니다.

예시

export const useUserStore = defineStore('user', {
  state: () => ({ name: '' }),
  actions: { setName(name) { this.name = name } }
})

기술 포인트

  • 언제 Pinia를 쓰나?부모-자식 관계를 넘어 여러 화면이 공유하는 상태가 필요할 때 사용합니다.
  • local state와 차이는?컴포넌트 내부 상태는 해당 컴포넌트 책임이고, Pinia는 앱 전역 또는 도메인 단위 공유 상태에 적합합니다.
