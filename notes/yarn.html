패키지 매니저

Yarn이란 무엇인가?

Yarn은 JavaScript/Node.js 생태계의 패키지 매니저입니다. 안정적인 lockfile, 빠른 설치, workspace 지원을 통해 프론트엔드와 모노레포 프로젝트에서 자주 사용됩니다.

간단한 예시

yarn install
yarn add axios
yarn dev
yarn build

주요 개념

개념설명
yarn.lock설치된 의존성 버전을 고정해 재현성을 높입니다.
workspace여러 패키지를 하나의 저장소에서 함께 관리합니다.
scriptspackage.json의 dev, build, test 같은 명령을 실행합니다.
BerryYarn 2+ 계열로 PnP 등 새로운 기능을 제공합니다.

기술 포인트

  • Yarn과 npm의 공통점은?둘 다 package.json을 기준으로 의존성을 설치하고 scripts를 실행하는 패키지 매니저입니다.
  • yarn.lock은 왜 중요한가?같은 프로젝트를 설치할 때 동일한 패키지 버전 조합을 재현하게 해 환경 차이를 줄입니다.
