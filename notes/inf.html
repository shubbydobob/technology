Operating System

.inf란 무엇인가?

Windows가 장치와 드라이버를 연결하고 설치 절차를 수행할 수 있도록 설명하는 설정 정보 파일입니다.

운영체제

.inf란 무엇인가?

.inf 파일은 Windows에서 장치 드라이버, 시스템 구성 요소, 설치 정보 등을 설명하는 Setup Information File입니다.

핵심 개념

항목설명
정식 의미Setup Information File
파일 형식텍스트 기반 설정 파일
주 사용처Windows 드라이버 설치, 장치 인식, 시스템 구성 요소 설치
실행 파일 여부직접 실행되는 프로그램이 아니라 Windows 설치 관리자나 Plug and Play 관리자가 해석합니다.
대표 위치C:\Windows\INF

왜 필요한가?

Windows는 새 장치를 발견했을 때 장치 자체만 보고 어떤 드라이버를 설치해야 하는지 알 수 없습니다. .inf 파일은 하드웨어 ID와 드라이버 파일을 연결하는 설명서 역할을 합니다.

설치 흐름

flowchart LR A[새 하드웨어 연결] --> B[Windows Plug and Play 감지] B --> C[하드웨어 ID 확인] C --> D[.inf 파일 검색] D --> E{ID가 일치하는가?} E -- 예 --> F[드라이버 파일 설치] E -- 아니오 --> G[알 수 없는 장치 또는 수동 설치 필요] F --> H[장치 사용 가능]

.inf 파일이 담는 정보

정보예시역할
제조사 정보Intel, NVIDIA, Realtek드라이버 제공자 식별
지원 OSWindows 10, Windows 11설치 가능한 운영체제 제한
하드웨어 IDPCI\VEN_8086...장치와 드라이버 매칭
복사할 파일.sys, .dll, .cat실제 드라이버 구성 파일 지정
레지스트리 설정서비스 등록, 장치 옵션Windows 동작 설정
디지털 서명 정보.cat 파일 참조드라이버 신뢰성 검증

관련 파일 비교

파일역할
.inf설치 절차와 장치 매칭 정보를 설명합니다.
.sys커널 모드 드라이버 본체인 경우가 많습니다.
.cat드라이버 패키지의 디지털 서명 카탈로그입니다.
.dll드라이버나 장치 유틸리티가 사용하는 라이브러리입니다.
.exe별도 설치 프로그램 또는 유틸리티일 수 있습니다.

면접 포인트

  • .inf는 실행 파일인가?아닙니다. Windows가 해석하는 텍스트 기반 설치 정보 파일입니다.
  • 드라이버 설치에서 역할은?하드웨어 ID와 드라이버를 매칭하고 설치 절차를 정의합니다.
  • .inf만 있으면 드라이버가 동작하는가?아닙니다. 실제 드라이버 파일인 .sys 등이 함께 필요합니다.
