slot이란 무엇인가?

부모 컴포넌트가 자식 컴포넌트의 특정 영역에 콘텐츠를 끼워 넣게 하는 Vue 기능입니다.

Vue3

slot이란 무엇인가?

slot은 컴포넌트 내부 레이아웃은 자식이 제공하고, 실제 콘텐츠는 부모가 전달할 수 있게 하는 기능입니다. 공통 카드, 모달, 테이블 컴포넌트에 자주 사용합니다.

예시

<BaseCard>
  <h2>제목</h2>
  <p>내용</p>
</BaseCard>

// BaseCard 내부
<slot />

기술 포인트

  • named slot은?header, footer처럼 여러 영역에 이름을 붙여 콘텐츠를 나눠 전달하는 방식입니다.
  • slot이 필요한 이유는?컴포넌트 재사용성을 높이면서 콘텐츠 구성은 부모가 결정하게 할 수 있습니다.
