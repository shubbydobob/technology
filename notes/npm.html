패키지 매니저

npm이란 무엇인가?

npm(Node Package Manager)은 Node.js와 함께 가장 널리 쓰이는 패키지 매니저입니다. 프로젝트의 package.json을 기준으로 의존성을 설치하고, 개발 서버 실행이나 빌드 같은 스크립트를 실행합니다.

간단한 예시

npm install
npm install axios
npm run dev
npm run build

주요 파일

파일역할
package.json프로젝트 이름, scripts, dependencies 정보를 담습니다.
package-lock.json설치된 의존성의 정확한 버전을 고정합니다.
node_modules실제 설치된 패키지들이 저장되는 폴더입니다.

기술 포인트

  • npm install은 무엇을 하나?package.json과 lockfile을 기준으로 필요한 패키지를 내려받아 node_modules에 설치합니다.
  • package-lock.json은 왜 필요한가?팀원과 배포 환경에서 같은 의존성 버전을 설치하게 해 재현성을 높입니다.
