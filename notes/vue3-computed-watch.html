Vue3

computed와 watch 차이

computed는 의존 상태가 바뀔 때 다시 계산되는 캐시된 파생 값입니다. watch는 특정 값의 변화를 감지해 API 호출, 로컬 저장, 라우팅 같은 부수 효과를 실행합니다.

예시

const fullName = computed(() => first.value + ' ' + last.value)

watch(searchKeyword, (value) => {
  fetchList(value)
})

비교

구분computedwatch

반환값부수 효과

기술 포인트

  • API 호출은 어디에?대부분 watch가 적합합니다. computed는 값 계산에 집중해야 합니다.
  • computed가 좋은 경우는?목록 필터링, 포맷팅, 합계처럼 상태에서 계산되는 값을 화면에 보여줄 때 좋습니다.
