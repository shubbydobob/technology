Frontend

props와 emit이란 무엇인가?

props는 부모가 자식에게 데이터를 전달하고, emit은 자식이 부모에게 이벤트를 알리는 방식입니다.

Vue3

props와 emit이란 무엇인가?

props는 상위 컴포넌트에서 하위 컴포넌트로 내려가는 입력값입니다. emit은 하위 컴포넌트가 상위 컴포넌트에 변경이나 액션을 알리는 이벤트입니다.

예시

const props = defineProps({ title: String })
const emit = defineEmits(['save'])
emit('save', form.value)

기술 포인트

  • props를 직접 수정하면?단방향 데이터 흐름을 깨뜨리므로 직접 수정하지 않고 emit으로 부모에게 변경을 요청합니다.
  • 왜 emit을 쓰나?자식 컴포넌트의 내부 이벤트를 부모가 제어할 수 있게 해 컴포넌트 책임을 분리합니다.
