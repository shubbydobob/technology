Frontend

v-if와 v-show 차이

v-if는 DOM 생성과 제거를 제어하고, v-show는 CSS display로 표시 여부만 바꿉니다.

Vue3

v-if와 v-show 차이

v-if는 조건이 false면 DOM 자체를 만들지 않거나 제거합니다. v-show는 DOM은 유지하고 display 스타일만 전환합니다.

비교

구분v-ifv-show
DOM생성/제거항상 존재
초기 비용조건 false면 낮음항상 렌더링
전환 비용높을 수 있음낮음

기술 포인트

  • 자주 토글하면?v-show가 유리한 경우가 많습니다.
  • 권한에 따라 아예 렌더링하지 않으려면?v-if가 적합합니다.
