Vue Router란 무엇인가?

Vue SPA에서 URL 경로와 컴포넌트를 연결하고 화면 전환을 관리하는 공식 라우터입니다.

Vue3

Vue Router란 무엇인가?

Vue Router는 URL에 따라 어떤 컴포넌트를 보여줄지 결정하는 Vue 공식 라우팅 라이브러리입니다. 중첩 라우트, 동적 파라미터, 네비게이션 가드를 지원합니다.

예시

const routes = [
  { path: '/users/:id', component: UserDetail }
]

기술 포인트

  • 동적 라우트란?/users/:id처럼 URL 일부를 파라미터로 받아 같은 컴포넌트로 다른 데이터를 보여주는 방식입니다.
  • Navigation Guard는?페이지 진입 전 인증이나 권한을 검사하는 라우터 훅입니다.
