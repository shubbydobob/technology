Vue3

양방향 바인딩과 v-model

v-model은 입력 값과 상태를 양방향으로 연결하는 문법입니다. input 값이 바뀌면 상태가 바뀌고, 상태가 바뀌면 화면 값도 갱신됩니다.

예시

const name = ref('')

<input v-model="name" />

컴포넌트 v-model

// 자식 컴포넌트
const model = defineModel()
model.value = 'new value'

기술 포인트

  • v-model의 내부 동작은?값 prop과 update 이벤트 조합을 간단한 문법으로 감싼 것입니다.
  • 언제 조심해야 하나?복잡한 폼에서는 검증, 초기화, 서버 동기화 흐름을 명확히 나누는 것이 좋습니다.
