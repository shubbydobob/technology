Vue3

Vue 생명주기란 무엇인가?

생명주기는 컴포넌트가 생성, 마운트, 업데이트, 언마운트되는 단계입니다. Vue3 Composition API에서는 onMounted, onUpdated, onUnmounted 같은 훅을 사용합니다.

대표 훅

훅시점
onMountedDOM에 컴포넌트가 붙은 뒤
onUpdated반응형 상태 변경으로 DOM이 갱신된 뒤
onUnmounted컴포넌트가 제거된 뒤

기술 포인트

  • API 호출은 어디서?초기 화면 데이터 조회는 보통 onMounted 또는 라우트 진입 시점에 처리합니다.
  • 정리 작업은?타이머, 이벤트 리스너, 구독 해제는 onUnmounted에서 처리합니다.
