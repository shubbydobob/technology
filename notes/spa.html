Frontend

SPA란 무엇인가?

SPA(Single Page Application)는 페이지 이동 때마다 서버에서 새 HTML을 받는 대신, 클라이언트 JavaScript가 필요한 데이터만 받아 화면을 갱신하는 방식입니다.

흐름

초기 접속: index.html + JS 번들 다운로드
이후 이동: Vue Router가 컴포넌트 전환 + API 데이터 요청

기술 포인트

  • SPA의 장점은?페이지 전환이 빠르고 앱 같은 사용자 경험을 만들기 쉽습니다.
  • 단점은?초기 로딩 비용, SEO, 인증 라우팅, 번들 크기 관리가 중요해집니다.
