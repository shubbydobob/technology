백엔드

서블릿이란 무엇인가?

서블릿(Servlet)은 자바 웹 애플리케이션에서 HTTP 요청을 받고 응답을 만드는 서버 측 컴포넌트입니다. Spring MVC의 DispatcherServlet도 서블릿 기반으로 동작합니다.

흐름

Browser -> WAS(Tomcat) -> Servlet -> Service Logic -> Response

역할

역할설명
요청 수신HttpServletRequest로 요청 정보를 받습니다.
응답 생성HttpServletResponse로 응답을 작성합니다.
생명주기컨테이너가 생성, 초기화, 호출, 종료를 관리합니다.
스레드 처리WAS가 요청마다 스레드를 할당해 처리합니다.

기술 포인트

  • 서블릿 컨테이너란?Tomcat처럼 서블릿 생명주기와 요청/응답 처리를 관리하는 실행 환경입니다.
  • Spring MVC와 관계는?Spring MVC는 DispatcherServlet을 중심으로 요청을 Controller에 분배합니다.
