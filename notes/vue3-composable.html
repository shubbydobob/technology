Composable이란 무엇인가?

Vue Composition API를 이용해 상태와 로직을 함수로 분리하고 재사용하는 패턴입니다.

Vue3

Composable이란 무엇인가?

Composable은 useFetch, useModal처럼 반응형 상태와 함수를 묶어 재사용 가능한 함수로 만든 것입니다. 화면 로직을 컴포넌트 밖으로 분리할 때 사용합니다.

예시

export function useToggle(initial = false) {
  const opened = ref(initial)
  const toggle = () => opened.value = !opened.value
  return { opened, toggle }
}

기술 포인트

  • Composable의 장점은?컴포넌트에서 반복되는 상태 로직을 함수로 분리해 테스트와 재사용이 쉬워집니다.
  • 주의할 점은?전역 싱글턴 상태인지 호출마다 새 상태인지 의도적으로 설계해야 합니다.
