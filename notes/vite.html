Vite란 무엇인가?

빠른 개발 서버와 최적화된 빌드를 제공하는 현대 프론트엔드 빌드 도구입니다.

Frontend

Vite란 무엇인가?

Vite는 개발 중에는 native ES modules 기반으로 빠른 dev server를 제공하고, 배포 빌드에서는 Rollup 기반으로 번들을 최적화하는 빌드 도구입니다.

특징

특징설명
빠른 시작전체 번들링 없이 필요한 모듈을 브라우저가 요청합니다.
HMR수정된 모듈만 빠르게 갱신합니다.
빌드프로덕션에서는 Rollup으로 최적화된 번들을 생성합니다.

기술 포인트

  • Vite가 빠른 이유는?개발 서버에서 전체 앱을 매번 번들링하지 않고 ESM을 활용하기 때문입니다.
  • Webpack과 차이는?전통적 번들러 중심 개발 서버보다 시작과 HMR이 빠른 구조를 목표로 합니다.
