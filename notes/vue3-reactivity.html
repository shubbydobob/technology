Vue3의 반응형 시스템이란 무엇인가?

상태 변경을 추적해 관련 화면과 계산 값을 자동으로 다시 갱신하는 Vue의 핵심 구조입니다.

Vue3

Vue3의 반응형 시스템이란 무엇인가?

반응형 시스템은 데이터가 바뀌었을 때 그 데이터를 사용하는 화면이나 computed 값이 자동으로 다시 계산되게 하는 구조입니다. Vue3는 Proxy 기반으로 객체 접근과 변경을 추적합니다.

간단한 예시

const count = ref(0)
const doubled = computed(() => count.value * 2)
count.value++ // doubled와 화면이 함께 갱신됨

흐름

단계설명
track값을 읽을 때 어떤 effect가 의존하는지 기록합니다.
trigger값이 바뀌면 의존하던 effect를 다시 실행합니다.
render변경된 상태를 기준으로 화면을 갱신합니다.

기술 포인트

  • 반응형의 핵심은?상태를 읽는 시점에 의존성을 추적하고, 상태가 바뀌는 시점에 관련 작업을 다시 실행하는 것입니다.
  • Vue2와 Vue3 차이는?Vue2는 Object.defineProperty 기반이고 Vue3는 Proxy 기반이라 객체와 컬렉션 추적이 더 유연합니다.
